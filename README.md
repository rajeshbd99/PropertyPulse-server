# PropertyPulse (Backend)

Welcome to the **PropertyPulse** backend repository! This is the server side of our real estate platform built with Node.js, Express, and MongoDB to ensure secure and efficient data handling.

## **Features**
1. **Secure API**: Endpoints protected with authentication and role-based access control.
2. **Database**: Efficiently stores users, properties, reviews, and transactions in MongoDB.
3. **Role-Based Functionality**: Different APIs for users, agents, and admins.
4. **Wishlist Management**: Allows users to manage their wishlist via API.
5. **Agent Property Tracking**: Tracks property status for agents (available, requested, or sold).
6. **Admin Management**: Comprehensive APIs for admins to manage users, properties, and reviews.
7. **Scalable Architecture**: Built with modular and scalable design patterns.
8. **Error Handling**: Global error handling middleware for better debugging.
9. **Environment Variables**: Secure sensitive data with `.env` configuration.
10. **Real-Time Notifications**: Integrated with modern notification libraries for updates.

## **Tech Stack**
- **Backend Framework**: Node.js with Express.js
- **Database**: MongoDB
- **Authentication**: Firebase Authentication
- **Environment Variables**: dotenv
